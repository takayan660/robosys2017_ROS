# robosys2017_ROS
[LEDをつけて」と言うとAlexaがLEDをつけてくれる

https://github.com/sparkgene/echo をROS化してみたもの

## Install 
```bash
$ git clone https://github.com/takayan660/robosys2017_ROS.git
$ cd robosys2017_ROS
$ ./install.sh
```

## Pin
5 pin output

## Run
```bash
$ sudo python alexa-led.py
```
